# Jira Integration

Connect Agent Mantis with Jira to automatically create issues for test failures and link test results to tickets.

## Features

- **Auto-create issues** — Failed tests create Jira tickets with screenshots and logs
- **Link tests to stories** — Associate workflows with Jira tickets for traceability
- **Status sync** — Test pass/fail status reflected in Jira
