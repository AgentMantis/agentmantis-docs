# Write Your First Suite

Create a collection of related test workflows for your application.

## What is a suite?

A suite is a group of workflows that test a specific area of your application. For example:

- **Login suite** — sign-in, sign-up, password reset
- **Checkout suite** — add to cart, apply coupon, complete purchase

## Step 1: Plan your tests

List the user journeys you want to cover:

```
1. User can sign in with valid credentials
2. User sees error with invalid password
3. User can reset password via email
```

## Step 2: Create workflows

In the Portal, create one workflow per test case.

## Step 3: Organise in a project

Group related workflows under a single project for easy management.
