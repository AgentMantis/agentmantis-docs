# Portal Overview

The Portal is your central dashboard for managing Agent Mantis projects, environments, workflows, and test results.

## Key features

- **Project management** — Organise workflows by application or team
- **Environment configuration** — Manage URLs, variables, and secrets per environment
- **Run history** — View past runs with screenshots, videos, and detailed logs
- **Team collaboration** — Invite team members with role-based access
