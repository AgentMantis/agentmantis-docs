# Artefacts

Artefacts are the outputs generated during a test run.

## Types of artefacts

| Artefact | Description |
|---|---|
| **Screenshots** | Captured at each step and on failure |
| **Video** | Full browser recording of the run |
| **Trace** | Playwright trace file for debugging |
| **Logs** | Detailed execution logs |

## Accessing artefacts

Artefacts are available on the run detail page in the Portal.
