# Editor Overview

The Editor is where you create, edit, and refine test workflows using natural language.

## Features

- **Natural language input** — Describe test steps in plain English
- **Live preview** — See your test execute in real time
- **Step management** — Reorder, edit, or delete individual steps
- **Assertions** — Add verification checks to your workflows
